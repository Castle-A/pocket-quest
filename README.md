# 🎮 Pocket Quest — Application Mobile

> App iOS/Android — Gagner du temps sur les Social Media en accomplissant des tâches saines

## 📋 Concept

L'utilisateur **erspielt** (gagne) du temps d'utilisation de Social Media Apps en faisant des activités saines :
- Chores (tâches ménagères)
- Sport / Sorties
- Marche (pas)
- Sommeil

Plus tu es sain, plus tu peux scroller. 🧠✨

## 🎯 Objectif

Créer une app qui rend la vie saine **fun** et **addictive** (dans le bon sens), avec un design mignon type Tamagotchi.

## 📊 Business Model

- **Freemium** — Version de base gratuite
- **Premium** — Abonnement auto-expirant (rappel à l'expiration)
- **New Abo** — Push notification quand expire → motivation re-call-to-action
